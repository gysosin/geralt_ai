# Product Guidelines: GeraltAI (New UI)

## 1. Core Principles
*   **Professional & Trustworthy:** The interface must instill confidence in the AI's outputs and system reliability.
*   **Modern & Sleek:** Utilize contemporary design patterns, clean layouts, and subtle animations (via Framer Motion) to create a high-end feel.
*   **Friendly & Helpful:** Interactions should be intuitive and supportive, guiding the user toward their goals.

## 2. Design Standards
*   **Minimalism:** Avoid clutter. Every element should have a clear purpose. Focus on whitespace and typography to define structure.
*   **Accessibility:** Adhere to WCAG 2.1 Level AA standards. Use semantic HTML and ensure high color contrast.
*   **Performance:** Optimize for fast initial loads and smooth interactions. Minimize bundle sizes and use efficient rendering patterns in React.
*   **Theming:** Full support for both **Dark and Light modes**, respecting user system preferences or allowing manual overrides.

## 3. Communication Style (Prose)
*   **Tone:** Senior Engineer - direct, concise, and technically accurate.
*   **Precision:** Avoid fluff. Use technical terminology correctly when appropriate for the context (especially for AI/Data science features).
*   **Clarity:** Ensure that system messages and AI responses are unambiguous and actionable.

## 4. Interaction Guidelines
*   **Feedback:** Provide immediate visual feedback for user actions (loading states, success/error messages).
*   **Consistency:** Maintain uniform spacing, sizing, and styling across all components using Tailwind CSS or standard CSS variables.
