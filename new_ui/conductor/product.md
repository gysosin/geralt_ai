# Product Guide: GeraltAI (New UI)

## Vision
To build a comprehensive, modern, and visually engaging frontend for the GeraltAI platform, empowering users to interact with advanced AI agents, visualize data insights, and manage intelligent workflows seamlessly.

## Target Audience
*   **Data Scientists & AI Engineers:** For configuring models, managing RAG pipelines, and monitoring system performance.
*   **Business Analysts:** For accessing data visualizations and deriving actionable insights from AI-processed information.
*   **General End-Users:** For intuitive conversational interactions with AI bots and assistants.

## Core Features (Inferred)
*   **Interactive Chat Interface:** A responsive and fluid conversational UI for interacting with AI agents (powered by Gemini/Mistral/OpenAI).
*   **Analytics Dashboard:** Visual representations of data and usage metrics using Recharts.
*   **Responsive Design:** A polished, mobile-friendly interface built with React and modern styling.
*   **State Management:** Efficient global state handling using Zustand for a smooth user experience.
*   **Visual Polish:** Smooth transitions and animations utilizing Framer Motion.

## Primary Goals
1.  Provide a robust and user-friendly entry point to the GeraltAI ecosystem.
2.  Facilitate seamless interaction with underlying AI services and document collections.
3.  Enable real-time visualization of complex data and AI outputs.
