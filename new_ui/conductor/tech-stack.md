# Tech Stack: GeraltAI (New UI)

## Frontend
*   **Core Library:** React 19
*   **Build Tool:** Vite (High-performance development and build environment)
*   **Language:** TypeScript (For type safety and improved developer experience)
*   **State Management:** Zustand (Lightweight, efficient global state handling)
*   **Routing:** React Router 7 (Modern, component-based navigation)
*   **Styling:** Tailwind CSS (Utility-first CSS for rapid, consistent UI development)
*   **Icons:** Lucide React (Comprehensive, consistent icon set)
*   **Animations:** Framer Motion (Powerful declarative animations and gestures)
*   **Data Visualization:** Recharts (Composable charting library for React)
*   **HTTP Client:** Axios (Promise-based client for API interactions)

## Development & Infrastructure
*   **Runtime:** Node.js
*   **Package Manager:** npm
*   **Type Checking:** TypeScript Compiler (tsc)
